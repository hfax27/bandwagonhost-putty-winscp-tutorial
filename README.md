## PuTTY怎么连搬瓦工

### 第一步：下载PuTTY

直接到PuTTY官网下载最新版本，选择对应Windows位数的安装包（一般选64-bit）。下载完双击安装，或者直接运行绿色版exe都行。

### 第二步：填写连接信息

打开PuTTY，你会看到一个配置界面，主要填这几项：

- **Host Name（or IP address）**：填你的服务器IP地址
- **Port**：填KiwiVM面板里看到的那个5位数SSH端口，**不要填22**
- **Connection type**：选SSH（默认就是SSH，确认一下）
- **Saved Sessions**：给这个连接起个名字，比如"搬瓦工"，然后点Save保存下来，以后直接双击就能连，不用每次重新填

填好后点右下角**Open**。

### 第三步：登录

第一次连接会弹出一个安全提示框，问你是否信任这台服务器，点"是（Y）"继续。

然后出现黑色终端界面，提示 `login as:`，输入 `root`，回车。

接着提示 `password:`，输入root密码。**注意：输密码的时候屏幕上什么都不显示，这是正常的**，不是坏了。密码别乱按，建议先把密码复制好，然后在PuTTY界面里鼠标右键一下（PuTTY里右键=粘贴），再按回车。

出现 `[root@... ~]#` 这样的提示符，就表示登录成功了。

**根密码在哪找？** 同样在KiwiVM面板里，也可以通过"Root Password Modification"功能重置密码。

## WinSCP怎么连搬瓦工

### 第一步：下载安装WinSCP

到WinSCP官网下载安装包，安装完打开。

### 第二步：新建连接

打开WinSCP后，会自动弹出登录对话框（或者点左上角"新建站点"）：

- **文件协议**：选SFTP（默认）
- **主机名**：填服务器IP地址
- **端口号**：填KiwiVM里的那个5位数SSH端口
- **用户名**：root
- **密码**：填root密码

填好后点**保存**，把"保存密码"勾上，方便下次直接登录。

### 第三步：登录和使用

点**登录**，同样第一次会弹出信任提示，点是。

进去之后，左边是本地电脑的文件，右边是服务器上的文件目录。拖拽就能传文件，右键文件可以选择用内置编辑器直接编辑，改完保存自动上传。

WinSCP还有个很实用的功能：界面上方有个"打开终端"按钮（图标是PuTTY的样子），点击可以直接打开PuTTY并自动继承当前的连接信息，不用重新填一遍。

## 常见问题排查

**连接超时/Connection refused**

检查端口号有没有填错。搬瓦工是5位数随机端口，不是22。另外确认VPS是否开机（KiwiVM面板里看状态）。

**密码错误**

密码区分大小写。建议在别的地方先把密码打出来确认正确，再复制到PuTTY里右键粘贴。如果实在不确定密码，去KiwiVM面板里重置。

**PuTTY输入密码没反应**

不是卡住了，是正常的——输密码时屏幕本来就不显示任何字符。输完直接回车就行。

**WinSCP上传文件很慢**

和你的网络到搬瓦工机房的连接质量有关。可以在KiwiVM里尝试切换到延迟更低的机房。
